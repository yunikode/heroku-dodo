# Heroku Todo API

Running on [Heroku](https://fierce-falls-todo.herokuapp.com)

First flight of the Dodo

```
yarn install
```

Server Up!

```
node server.js
```

Get all todo items

```
curl localhost:3000/todos
```

Get individual item number 1

```
curl localhost:3000/todos/1
```
